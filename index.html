Name Description Ages Location Cost
CODE.org Code.org is a non-profit organization and eponymous website led by brothers Hadi and Ali Partovi that aims to encourage people, particularly school students in the United States, to learn computer,science. Code.org features coding games, learning exercises and educational resources for all ages. Ages:all Online Free
CODECADEMY Codecademy is an online interactive platform that offers free coding,classes in 8 different programming languages including Python, PHP, jQuery, JavaScript, AngularJS, and Ruby, as well as markup languages HTML and CSS. Ages:all Online Free
GIRL DEVELOP IT Girl Develop It is an international organization with local chapters in Buffalo and Rochester that provides affordable and accessible programs to women who want to learn software development through mentorship and hands-on instruction. Ages: 18+ Buffalo, NY Varies
Buffalo Game Space Buffalo Game Space is a 501(c)(3) non-profit dedicated to helping people make games by offering the educational, mentorship, and equipment resources to both professional and aspiring game developers. Ages:All Buffalo, NY Varies

Authors and Contributors

As this is a grassroots effort, you are encouraged to contribute resources, organizations, and events that provide educational or vocational training in the broad area of technology that are available to residents in the Buffalo area. This page was created and is currently maintained by David Brenner (@davidbrenner). You can submit these resources via pull requests/issues on the github page](https://github.com/davidbrenner/buffalocodes.org/) or via email.
